Build it with this generic app!

This app will include some common capabilities that all modern web apps use. The app may have some simple example features to demonstrate functionality.

Goals

The current goal is to incorporate each of the following frameworks and libraries:

  • Vagrant with virtual box integration
  • Flask app with jinja2 template and index page
  • Simple jquery request for hello world json
  • Simple Rest interface for an example object + jquery interaction
  • Twitter bootstrap for UX
  • Berkshelf
  • Deploy with chef server
  • MySQL cookbook
  • Python dev cookbooks
  • Flask cookbook
  • Simple rest CRUD API for DB object
  • UI for CRUD-ing simply object
  • Deploy with app server nginx and wsgi
  • Auth with some tools such as facebook or linked in
  • Read a simple text file from a specific path (use README.md)

Getting Started:

  • Install Vagrant
  • Install Virtual Box
  • vagrant up
